# Talk To Data v3 - Frontend

Next.js frontend for Talk To Data.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts (for visualizations)

## Project Structure

```
app/
├── components/ui/     # shadcn/ui components
├── components/        # Custom components
├── hooks/             # React hooks
├── lib/               # Utilities
├── types/             # TypeScript types
├── page.tsx           # Main page
├── layout.tsx         # Root layout
└── globals.css        # Global styles
```
